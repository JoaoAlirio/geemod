# Access to the application
## web version
<a href="https://geemod.projects.earthengine.app/view/geemod">launch geeMod</a>  

## code editor version
<a href="https://geemod.projects.earthengine.app/view/geemod">import geeMod scripts directory</a>  
the directory containing the geeMod scripts will be imported into the code editor environment of your gee account (see GEE > Scripts > Reader)
